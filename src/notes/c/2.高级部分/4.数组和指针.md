## 数组和指针

``` c
#include <stdio.h>

int main(){
  char str[] = "Hello World!";
  char *p = str;
  char p2 = str[0];
  printf("%c %c",*p,p2);  
}
```

发现 `p` 和 `p2` 的值是一样的，也就是说下标也就是一个简化后的指针

本身的 `str` 保存的就是 数组 的首个地址，指针也就也是这个地址

指针也是可以进行运算的

``` c
#include <stdio.h>

int main(){
  char str[] = "Hello World!";
  char *p = str;
  //            ↓ 这样就往后移动了一倍的对应类型大小
  printf("%c",*(p+1));  
}
```

这样我们就知道了 `*(p+i)` 和 `str[i]` 的作用是相同的
