## 局部变量

```c
int main() {
  int a = 1;
  test();

  if(1) {
    int b = 2;
  }

  printf("%d",b); // <- 此处也无法访问到 b

  int c;
  for (c = 0; c < 10; c++) {

  }
  printf("%d",c); // <- 此处可以访问到 c
}

void test() {
  a++; // <- 无法访问到a,这是局部变量
}
```

## 全局变量

```c
int a = 20;

int main() {
  a += 1; // <- 可以访问到
  test();

}

void test() {
  a++; // <- 可以访问到
}
```
