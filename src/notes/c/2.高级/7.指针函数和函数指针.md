## 指针函数和函数指针

### 指针函数

::: warning
不要返回局部变量的指针!!!
函数运行结束，里面使用的内存就已经标记为可重新分配了，虽然值还在
:::

```c
#include <stdio.h>

int * test(int * a)
{
  return a;
}

int main()
{
  int a = 10;
  int * p = test(&a);
  printf("%d\n", *p);
}
```

### 函数指针

```c
#include <stdio.h>

int sum(int a, int b)
{
  return a + b;
}

int main()
{
  int (*p)(int, int) = sum;
  int result = (*p)(1,2);
  int result2 = p(1,2);
}
```
