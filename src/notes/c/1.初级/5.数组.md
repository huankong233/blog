## 创建

数据也可以留空，但是数组的长度是必须的
如果数据固定了，长度可以留空

```c
变量类型 变量名称[数组长度] = {数据}

// int arr[5] = {1,2,3,4,5};
```

右侧的数据是有顺序的，但是可以是跳跃的，比如

```c
int arr[10] = {1, 2, 3, [5] = 6, [8] = 9};
```

如果是多维数组的话就需要多次声明数组的长度，数组中数组的长度

```c
int arr[3][5] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

以此类推

## 访问

访问时只需要:

```c
arr[第几位]
```
