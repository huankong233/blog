## 函数列表

### 1.随机x位字符串

函数名: `getRangeCode`

函数参数：

| 字段          | 类型  | 说明       | 默认值    |
| ------------- | ----- | ---------- | ---------- |
| `len` | `Number` | 需要随机多少位字符串 | `6` |

### 2.生成随机整数

函数名: `randomInt`

函数参数：

| 字段          | 类型  | 说明       | 默认值    |
| ------------- | ----- | ---------- | ---------- |
| `min` | `Number` | 最小是多少 | `0` |
| `max` | `Number` | 最大是多少 | `1` |
