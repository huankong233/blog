大部分接口都可以直接在 `实例化对象` 上直接调用, 比如 发送信息 `send_msg`

此处的 `Structs` 在 [下一篇](./5.消息段构造器.md) 中有介绍

~~~ typescript
import { Structs } from 'node-napcat-ts'

await napcat.send_msg({
  user_id: 10001,
  message: [Structs.text("你好呀~")]
})
~~~
