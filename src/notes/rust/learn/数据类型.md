## 标量类型

- 整数
- 浮点
- 布尔
- 字符

### 整数类型

- 有符号以 `i` 开头
- 无符号以 `u` 开头

默认类型为 `i32`

### 浮点类型

- f32 —— 单精度
- f64 —— 双精度

默认类型为 `f64`

### 布尔类型

- `true`
- `false`

### 字符类型

- `char` —— 单个字符

## 复合类型

- 元组(Tuple)
- 数组(Array)

### 元组

用于将多个值放在一个类型里 固定长度 一旦声明无法改变

例子:

```rust
let tup: (i32, f64, u8) = (500, 6.4, 1);
```

获取值:

```rust
let (x, y, z) = tup;

println!("{} {} {}", tup.0, tup.1, tup.2);
```

### 数组

用于将多个值放在一个类型里 固定长度 一旦声明无法改变 每个元素类型必须一致

```rust
let array: [i32, 4] = [1, 2, 3, 4];
```
