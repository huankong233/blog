## 更新

### push_str()

把一个字符串切片附加到 `String` 上

### push()

把单个字符附加到 `String` 上

### +

用于连接字符串 但是会导致第一个变量被获取所有权 一般使用 [format!](#format)

~~~ rust
let s1 = String::from("Hello,");
let s2 = String::from("World.");

// 这里就导致了 s1 失去了所有权 无法再使用
// s2 因为是引用 所以还可以继续使用 
let s3 = s1 + &s2;
println!("{}", s3);
~~~

### `format!`

使用类似于 `println!` 而且不会获取所有权

~~~ rust
let s1 = String::from("Hello,");
let s2 = String::from("World.");

let s3 = format!("{}-{}", s1, s2);
println!("{}", s3);
~~~
