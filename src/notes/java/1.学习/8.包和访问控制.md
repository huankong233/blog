## 声明包

使用 `package` 关键字声明当前文件的位置 (IDEA 会自动管理)

## 导入包

使用 `import` 关键字导入包, 需要包含到最后一级类的名字

## 访问控制

一共有四种访问权限

|           | 当前类 | 同一个包下的类 | 不同包下的子类 | 不同包下的类 |
| --------- | ------ | -------------- | -------------- | ------------ |
| public    | ✓      | ✓              | ✓              | ✓            |
| protected | ✓      | ✓              | ✓              | ×            |
| 默认      | ✓      | ✓              | ×              | ×            |
| private   | ✓      | ×              | ×              | ×            |

如果使用了 `private` 定义了类的构造函数, 那么这个类就无法被直接实例化

除非编写一个静态方法去手动实例化类
