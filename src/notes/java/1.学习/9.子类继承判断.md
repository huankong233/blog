## 使用 instanceof 关键字

:::tip 在 `JAVA 16` 中新语法

```java
// 变量          子类    变量名
if(a instanceof Student student){
  // 可以省略强制类型转换的步骤
  student.study();
}
```

:::

```java
if(a instanceof Student){
  student = (Student) a;
  student.study();
}
```
