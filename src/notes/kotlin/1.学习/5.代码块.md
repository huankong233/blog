`kotlin` 和 `java` 中的循环也一样, 可以设置代码块来指定跳出的循环位置

```kotlin
fun main() {
    xxx@ for (i in 1..10) {
        for (j in 1..10) {
            //                         ↓ 这里不能有空格
            if (i == 5 && j == 5) break@xxx
            println("i=$i, j=$j")
        }
    }
}
```
