## 函数列表

### 1.格式化时间,将毫秒转换为 HH:MM:SS 的格式

函数名: `formatTime`

函数参数:

| 字段 | 类型     | 说明          | 默认值      |
| ---- | -------- | ------------- | ----------- |
| `ms` | `number` | 时间(单位:ms) | `undefined` |

### 2.判断是否为今天

函数名: `isToday`

函数参数:

| 字段        | 类型     | 说明   | 默认值      |
| ----------- | -------- | ------ | ----------- |
| `timestamp` | `number` | 时间戳 | `undefined` |

### 3.获取当前的时间 2023/6/26 09:46:39

函数名: `getTime`

### 3.获取当前的日期 2023-06-26

函数名: `getDate`

### 4.统计运行时间

函数名: `countRunTime`

函数参数:

| 字段   | 类型 | 说明           | 默认值      |
| ------ | ---- | -------------- | ----------- |
| `func` | `T`  | 需要运行的函数 | `undefined` |
