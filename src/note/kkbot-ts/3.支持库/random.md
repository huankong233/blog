## 函数列表

### 1.随机 x 位字符串

函数名: `getRangeCode`

函数参数:

| 字段  | 类型     | 说明                 | 默认值 |
| ----- | -------- | -------------------- | ------ |
| `len` | `number` | 需要随机多少位字符串 | `6`    |

### 2.生成随机整数

函数名: `randomInt`

函数参数:

| 字段  | 类型     | 说明       | 默认值 |
| ----- | -------- | ---------- | ------ |
| `min` | `number` | 最小是多少 | `0`    |
| `max` | `number` | 最大是多少 | `1`    |

### 2.生成随机浮点数

函数名: `randomFloat`

函数参数:

| 字段  | 类型     | 说明       | 默认值 |
| ----- | -------- | ---------- | ------ |
| `min` | `number` | 最小是多少 | `0`    |
| `max` | `number` | 最大是多少 | `1`    |
