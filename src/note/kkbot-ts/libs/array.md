## 函数列表

### 1.排序对象数组

函数名: `sortObjectArray`

函数参数:

| 字段       | 类型               | 说明     | 默认值      |
| ---------- | ------------------ | -------- | ----------- |
| `arr`      | `{[key:string]}[]` | 数组     | `undefined` |
| `property` | `string`           | 排序属性 | `undefined` |
| `sortType` | `"up" \| "down"`   | 排序方式 | `"up"`      |
