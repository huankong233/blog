## 例子

这里的 `r` 是不能正常打印出来的 因为在使用这个引用的时候 `x` 已经被销毁了

~~~rust
fn main() {
    let r;
    {
        let x = 5;
        r = &x;
    }

    println!("{}",r);
}
~~~

## 手动标注

~~~rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str{
    if x.len() > y.len(){
        x
    } else {
        y
    }
}
~~~
