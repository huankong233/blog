## 函数列表

### 1.注册对象属性到 `global` 上

函数名: `globalReg`

函数参数:

| 字段  | 类型     | 说明                         | 默认值      |
| ----- | -------- | ---------------------------- | ----------- |
| `obj` | `Object` | 需要注册到 `global` 上的属性 | `undefined` |
