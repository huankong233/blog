### 用途

1. 创建事件的[触发器](#触发器)

### 用法

#### 参数

| 字段          | 类型  | 说明       |
| ------------- | ----- | ---------- |
| `type`     | string | 事件类型     |
| `callback` | function | 回调     |
| `priority` | number | 优先级     |

#### 返回数据

无

### 触发器

1. 支持优先级 `100` 为最高，如果有特殊必须最高优先级，即使用 `101` 优先级即可
2. 如果函数返回 `quit` 就不会往后处理数据了，可以搭配优先级支持
