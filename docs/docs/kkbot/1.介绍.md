这就是个简单的QQ机器人的框架（我说的）实现

主要使用了这些库作为依赖:

- [go-cqhttp](https://github.com/Mrs4s/go-cqhttp)
- 用于提供qq的交互功能
- [@tsuk1ko/cq-websocket](https://www.npmjs.com/package/@tsuk1ko/cq-websocket)
- 用于链接go-cqhttp
- [go-cqwebsocket](https://www.npmjs.com/package/go-cqwebsocket)
- 用于生成cq码
